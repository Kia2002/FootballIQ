# FootballIQ Scout

> Natural language player scouting API — under active construction.

**FootballIQ Scout** is a production-quality ASP.NET Core 9 API that answers football scouting queries in natural language using a hybrid RAG pipeline over StatsBomb event data.

Ask: _"Find a left-back under 23 with 85%+ pass accuracy and high pressing intensity"_  
Get back: ranked player profiles, confidence scores, and a natural language explanation.
